# Бот «Клуб родителей ЭДпауер» — стартовый каркас

Квиз-воронка с персональной картой развития, оффером (все CTA ведут на сайт, оплаты в боте нет) и добивочными сериями A/B/C/D.

Это **рабочий скелет** для доводки и запуска в Claude Code. Логика и тексты на месте; Claude Code поможет запустить, проверить в Telegram и задеплоить.

## Файлы
- `bot.py` — квиз, оффер, обработчики кнопок.
- `texts.py` — все тексты, варианты ответов, подстановки в карту (менять только по согласованию с продуктом).
- `scheduler.py` — добивочные серии по расписанию.
- `db.py` — состояние пользователей (SQLite).
- `config.py` — настройки из `.env`.

## Запуск локально
1. Установить Python 3.11+.
2. `pip install -r requirements.txt`
3. Скопировать `.env.example` → `.env` и вставить:
   - `BOT_TOKEN` — от @BotFather (если токен где-то засветился — выпустите новый через `/revoke`).
   - `SITE_URL` — ссылка на страницу клуба.
   - `GIFT_URL` — ссылка на файл чек-листа (по желанию).
4. `python bot.py` — бот запустится, напишите ему `/start` в Telegram.

## Деплой 24/7
Любой простой хостинг (Railway, Render, VPS). Переменные из `.env` задать в настройках окружения хостинга. Бот работает на long-polling — внешний URL не нужен.

## Что доведено
- ✅ `on_resume` возобновляет квиз с последнего шага (по сохранённому `stage`), а не с начала.
- ✅ Проверены все 3 возрастные ветки × 4 боли и корректность подстановок в карте.
- ✅ Планировщик: серии A/B/C/D, приоритет C > B > D > A, лимит 1 сообщение в день, стоп при `converted=1`.

## Заметки для дальнейшей доводки
- Точный стоп серий при реальной оплате — повесить вебхук с сайта/CRM, который ставит `converted=1` (пока стоп держится на кнопке «Я уже вступил»).
- Тексты серий и тайминги — в `texts.py` и `scheduler.py`; тайминги в минутах.

## Безопасность
